新人王——草芥君的出山作，改编自《嗨，时间彼岸的我》同名轻小说，漫画很好的还原了原著的意境...
..2日、3日... 白天开始透过窗户就可以一直嗅出，远处四散的爆竹声... 不时簇拥着烟火...
不过，现在是夜 ...选择了这个时间... 抱歉没有读过原著... 写的是我自己的事...

————


故乡的白兰树... 那是无论何时，身在何处的每个夜晚都会在我的脑海里闪现出的一副水彩...
此刻我身在越洋的油船里，她们若隐若现... 我旅居美国念书，从那以后，离开故土已是4年...

——题记
（注：本文时间全是虚指，瞎掰的。）


逛过城隍庙，中午我们回到家... 一屋子的女人便聚集到厨房准备生火做饭。

我不会做饭，母亲便给了我篮子去买不够用的蛋饺和西红柿、500ML装可乐。

那是一间职工分配的大院，不处于城乡结合部，但也不是市中心。

我手里拎着从国外买的皮包，趁着四婶和姑妈坐在地上淘米的时候小声告了别，下楼 ...

楼下便是菜市... 姑妈家住在二楼。楼房有6层高，在那个时候她并不算矮，不可能被划入即将被拆迁的名单里。


姑妈家平时很少用冰箱，因为楼下就是菜市 ... 只有在省城的夏天，知了和蝈蝈不安的倾巢而出，酷暑难耐之时，
才会有人打开冰箱的电源，为我储存冰棒... 那时每年夏天我们家的保留节目，冰棒的批发商铺在菜市的尽头，旁边就是小学...

批发商的店里有一座巨大的可以容纳十几个人的超大冰箱...
据说周围十几里包括市区，所有的店都是从这里进货... 后来... 她搬到了不知道市区的哪个地方...


在我离开之后的时间里，冰箱终于因为长时间没有用而坏掉，不过现在她还在用，那是一个放鸡蛋的储藏柜。

而离菜市很近的另一个好处是，总是可以买到新鲜的蔬菜，频繁往来于城郊和市区的菜贩阿姨和农民伯伯，总是会先经过这里。


已经到了吃年夜（误：午）饭的时间... 常日里忙碌的菜市，今天却格外的安静，只有一地的菜叶和碎冰，
提醒着人们这里可能真正是一个菜市，一并诉说这里往日的繁荣与喧嚣。


好运的事是，我从路边唯一的一处卖菜的那儿和街边仅存的还在营业的店里买到了我要买的所有物体。
并且，蛋饺还是比平常优惠0.5倍的价格... 似乎大家都赶着卖完最后的存货而好回家过年。


卖菜的是一位14岁穿着花红棉袄的小女孩，看起来要更大一点，像十六岁，皮肤黝黑... .. 她手里剥着芫荽，不看我——我不认识她 ...
... 店铺坐落在街角，老板娘是一位40来岁的妇人...一直看我、并对我笑——她好像认识我 ? ...

（芫荽也就是香菜，原产地是地中海沿岸，芫荽是我们家一直惯用的称呼。。。）

我不情愿的从袖口里伸出另一支手，提着篮子回家并且走上楼，不是沿着刚才的路线。

可乐很重... 仿佛身体有些疲倦，想要瞌睡...

我把菜放在了一边，厨房现在站满了人，从炉灶生出的火，菜籽与水煎熬的声音，散发着淡淡的油烟。

我不喜欢这儿，我用围领半屏住呼吸，快步走到阳台。

...

...!

（正文开始）

（一）

「——你是谁？」 听到了轻微的脚步声，小女孩回过头看了我，咬着嘴唇，并说。

她像极了我认识的一个人，我的一个朋友。

但那是谁？我一时脑海里竟谁也回忆不起... 但我咬定这种「记忆」不是什么要紧的事，我有过经验。

「嗯嗯... 我是你姐 ... 」

我嘴里这样说，但是我也不确定我是她姐，我是根据以往旧历的新年，总会有这样那样的不认识的或远或近，或年长或幼小的亲戚家的小孩做出的判断。

小女孩缓慢的转过身子，动作比刚才要快 ... 她穿着一件白色的裙子，里面穿着红色的棉袄和白色丝绸衣服，脖子上系着鲜艳的红领巾。红白相间，衣服上绣着象征着我们家族符号的花瓣，袖口缝纫华丽的花纹图案。

那种花瓣，只在重大节日的时候我才会有印象，要等到过年时才会穿的一种布料。

...

我伸出一只手放在胸前，掀开外面的皮衣... 露出最里面的一件衣服...

这是和你身上一模一样的花瓣，我想，但没有说话，和几年前一样，我仍然不喜欢说话 ...

这证明我是她的姐姐，或者至少是近房的亲戚。 （推理错。。）

小女孩立刻放松了之前对我的戒备，喜露眉梢，对我笑，并做出了一个「表示不认识的手势」。

我偷偷的窃笑... 这个小孩是多么的容易骗，我心想，她们放松对一个人的警惕就像建立起对一个人的警惕那样的毫无道理。

我蹲下身子，用手掌摸了一下她的额头，问道：「你今年，多大？」

小女孩没有回答我，犹豫了一会儿，然后义正言辞的拒绝回答：「问年龄是对女士不礼貌的行为！」 ...

我没有办法，再问道：「你今年念幼儿园什么班？」

... 小女孩皱着眉头，纠正我说：「我六年级！」...

我惊异，因为这个小女孩的个头，顶多让人看上去觉得是小学一年级，我暗自责备，看来猜度的方法总是不正确。

...

（二）

我正准备道歉，小女孩自言自语道：「嗯嗯，我想吃草莓...」，突然她转过身子，那个角度听见了猫咪的声音，她蹲下身子，伸出两只掌心，

一只只有松树大小黑猫便跳进她的怀里。

小女孩抚平了猫身上的毛，等到小黑猫做出睡觉的姿势，才站起身子，抱着她望着阳台之外。

她似乎忘记了我这个人...

因为我正无聊，便也靠近栏杆，小女孩的目光则紧紧看着外面雪后的世界 ...

我便好奇的凑过去问她，「嗯嗯，告诉姐姐，你在，看什么？~」

...

「槐树。」

我朝着她目光所指向的地方望去，没有槐树，最近距离，只有一株白兰树的树桩，被人砍断了枝杈的地方系着尼龙绳，另一端绑在电线杆上晴天的时候用来晾晒衣服。

我摇头，她向远景处指着西面，喃喃的说道：「那是我们学校。」

哪里？我看不到，我的视线被完全的遮盖，那里建立起了一座巨大的家具中心，灰色的油漆墙体...

只有顶处还露出了一竖生锈的旗杆。


「——那是我们学校。」

... 我当然知道，那里便是菜市的尽头，楼下便是菜市，在往后的漫长时光里，我确实知道这其实是多么适宜居住的地方，没有大城市那种奔流不息的汽车，大型超市里挤满的人，一路螺旋飙升的房价地价和物价 ... 最高的步行速度和十足的慌乱感 ...

也没有居住在山区里那种物资的相对缺乏... 当然我希望生活在海边 ...

——如今那里已经没有人烟，再魔鬼的学校在这个时间也会让学生还家，校长也要回家过年。

附近不只这一所学校，然而这并不是读书的最佳去处，事实上，我后来发现，仅仅相隔半个老城区的地方就有这比这里好的多的学校。

这里是一个适合生存的地方，但没有很好的教育 ...

那是我以后才认识到的事情，我离开这走的越远，见到了更多的人和事物，对这个问题的认识就越深。

后来，有一次，初一的时候我回了母校，学校门口写着我名字的红榜还没有撕掉，贴了一年也已经有些褪色。

班主任看到了我回来，高兴对我嘘寒问暖，让我在她现在的班里作演讲，我紧张的半天说不出话。

老班便拉我到办公室里，带着我写下一份稿子，交给了另一个小女孩——她的课代表，一定要在红领巾广播站里朗诵 ...

...

我倒是很乐意写字。

.....

在到后来我在外地念书，每次回家里，在这个小地方总会引起人群的围观，长辈们会用我来作为劝说子女用功读书的例子...

而在1999年以前，我并不认识 ...

..

楼下的爆竹声打破了我短暂的回忆，或者，也许是同时打破了我们两个人...

我们同时朝声音传来的地方望去，那个方向上，几个男生不断奔跑着，后面托着一只似乎怎么也飞不起来的燕子风筝。

女孩并不像我，她丝毫不能隐藏这种因看到别人的风筝坠落在地上而产生的隐隐的狂喜，这点从她嘴角那种狡黠的笑容便能猜测完全。

那种傲慢的表情，深深的酒窝，那副迷人的眼睑像极了... 我小时候？

我看着她的眼睛，乌黑而修长但是柔然而下垂的睫毛、问她...

「——小同学你想念那所中学？」

她似乎对这个问题毫不关心，继续沉溺在她的想象里。

我换了语气，重新问...

「——你长大以后想做什么事情？」

她埋下头，似乎思考片刻，突然转过头看我。

...

她看着我，我不知所措 ...

「流这义人的血，罪不在我，你们承担 ...」

... 「这句子——姐姐，你去过... 国外么？欧洲和美国？... 」

「嗯！去过的。」

她快速的蹲下身子，小猫从她的怀里跳到笼子里 ...  ...

「那些地方会有些什么呢？ ...」 她欢快的拍手，「如果可以，我还想去新加坡、札幌、斯德哥尔摩 ...」

我一时不知道该怎么回答，不知道编些什么糊弄她过去...

她让我记起了不妙的回忆 ... 那些留学期间发生的所有事情，我可能要写一本完整的书才能和她解释清。

又是一声爆竹，在附近爆炸。然而爆竹声比起我，已经不能再激起她的兴趣，我也已经过了欣赏烟火的年龄。

然而我还是朝那个方向望去，没有一个人，只有天空中涌现的四散而开一缕灰烟，和在地上的一纸红屑，辐射般的降落。

我表情木然，在这个家族中属我去过的城市最多，然而我已不愿再告诉她。

——既然她的志向在国外，我不愿告诉她我失望的见闻，那么一切，自然会留给她以后体会。

这是过年，从同一栋大楼传来连续不断的爆竹声，火光坠落到楼下，挂在一颗古树之上 ...  烧灼 ...

她双手捂着耳朵，我睁开眼睛 ...

直到大片的火光褪去，只是在废墟里仍然残余着零星的爆破声。

这便是旧历新年，一会儿会有男生呼啸而过，拎着一塑料袋的烟火——其中一种是会发射出去的火箭。

点燃之后，他们会向前推进，发出一种尖锐的泣声——如少女死前的悲鸣！直到消失在低矮的云层之中，砰的一声在这个世界里灰飞烟灭。

像这样的声音——「ふゅふゅふゅふゅ——ぼう！」

再过一会儿，便会有男生走到我的城下喊我的名字。


（三）

「——我们下去吧，一起去找人玩。」我说。

...

..

远处吹过一阵寒冷的风，卷起地上的绯红。不知道是凋落的花瓣还是爆竹的皮囊。

一丝寒意随风渗进我的袖口，一种消极厌世的感情油然而生。

「嗯不... 」

她抬起头，眼生里滞留着不满与无奈。

「其实我没有什么自由的... 你自己去就好...」

又补充了一句...

「我永远在家...」

房间里传来油水相遇的声音，火焰高涨数丈，嘻嘻哈哈的笑声，女人们忙着做饭聊天，男人们则围桌在客厅里打麻将，看电视，嗑瓜子。

而我不会做饭，因而只好到房里，而我同样不会嗑瓜子，因而只好到阳台——幸而还有阳台，这里是家里现在留给我唯一的安静的地方...

这是我的塔楼，城堡，无数的午后放学，无数的时间我坐在阳台，望着楼下奔跑的少男少女，一年四季，景随境迁，叶生花落，我望着他们嬉笑——拍手、跳皮筋，...

——快，快别写了！

不，我要写，我要「反芻」！...

...


我嗔视着窗外，我看到那株「槐树」被人砍下，我看到了现场的始末！

庭外，那原本中通相连的谷道，我骑单车从那里曾经经过，如今被住在楼下的人锁上铁门，围成自家田地。

那些人现在又身在何处？。。。

。。。

我记起了弗吉尼亚大学大门上刻着的希腊文 ... 我不知道，从何时开始

我将 「因真理，得自由，以服务」 作为自己一生的座右铭，这种感情从何时开始就已经淡忘，以至于无存...

... 以至于——那锁在房中面壁读书往来于教室时间回廊里的树的数十年间的晨昏层层密密——而当我终于有能力踏上这些魂牵的理想之土地，然而我却成为了一名活死人不是？... 从何时开始，我便开始求得追求自己的生活...我同 Paul 谈了两年的恋爱 ... 哦 ... 那段时间...那段时间里..

人们的轻蔑、鄙视和恼火，物质生活的不舒适或舒适... 对我越发的重要。

我心悸，喘气，不能自已。

我再看向小女孩，一把推开了她的身体，扯下了她胸前的红领巾。

撕成碎片 ...

（四）

我没能把它撕成碎片——尽管我用尽了全身气力——我呆呆的站在阳台的门口，没有阳光。

我用力过猛，扯断了女孩裙子的一侧的布带，一侧的连衣裙哗的从肩膀落下而去，露出里层的衣服，花瓣纹饰。

她没有反应过来...

而当她反应过来的时候，一把夺过了我手里撕了一半的一张红领巾。

两只手把她紧紧的攥在手心里，诉说着我... 似乎在以她的年龄诉说着我刚才无限的罪孽，不可饶恕，无论什么样的刑法也不够！

我失去了刚才抢来的东西... 不知道该说什么...

我低头，俯身，心如刀绞，隐隐作痛。——我知道了这里发生的一切！

我知道我再也不能控制住眼前的景象，反正都是红色的碎片。

我一只手抚上滚烫的额头，遮住一只眼睛，无力的站在原地...

眼泪从顺着眼睑流出，沾湿了眼睫毛，流过手腕。

我把头埋在围领里，我哭出声音，尽管屋里的人什么还都不知道。

...

此刻，那个小女孩却向我这边走来，她放松了刚才的举动，把刚才那张写满了我的罪恶的红巾“还”给了我。

她还没有哭，但我知道她已经哭过了 ...

「——其实，我什么都知道，不用你说...」

「——我什么都知道，不用你说什么的...」

她凑到我的耳边，用手拭去我的眼泪，我用哭红的眼圈望着她 ...

她说了什么... 她接触到了我，身体散发出微茫的蓝光——然后暖化成一种名为「回忆」碎片，缓慢的融化在我的视线里。

阳台外，开始下雨，雨水落的很快，像是夏天，她们混杂在土地里，不知道是冰还是雨。


（五）

饭烧好了，大人叫我回房吃饭...
