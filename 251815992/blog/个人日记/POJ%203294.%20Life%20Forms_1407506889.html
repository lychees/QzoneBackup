POJ 3294. Life Forms
给定n个字符串，求出一个尽可能长的子串，使得该子串至少出现在n/2+1个给定的串中。
http://vjudge.net/problem/viewProblem.action?id=17349



... 数据还是弱。

SA + 二分。。

http://vjudge.net/problem/viewSource.action?id=843910


SAM + DP
http://vjudge.net/problem/viewSource.action?id=2611971
。。
我们把所有字符串接起来（中间用 “$”）隔开构建 SAM。。。。。
利用 fail 树、我们可以 DP 出每个状态表示的子串，在各个字符串中出现的次数。。
。。于是也就得到了合法子串的最大长度。。

。。。但是这样得到的长度有可能受到 "$" 的影响而偏大。。（或者你每次都用不同的字符串隔开。。但是那样需要开 map。。）
。。因此我们尝试拼写所有子串。。

void dfs1(int u = 0, int k = 0){
if (cnt[u].count() <= n/2) return;
checkMax(max_len, len[u]);

REP(c, Z-1) if (len[v] == len[u] + 1){
buf[k] = 'a'+c, dfs1(v, k+1);
}
}

这样可以得到真实的最大值。。再 dfs 一次输出即可。。。

void dfs2(int u = 0, int k = 0){
if (cnt[u].count() <= n/2) return;
if (k == max_len){
buf[k] = 0;
puts(buf);
return;
}

REP(c, Z-1) if (len[v] == len[u] + 1){
buf[k] = 'a'+c, dfs2(v, k+1);
}
}
