嗯...这篇呢应该会包含两个主题，我要分开写。。。（也有可能是三个。。）

。。。

第一部分。。。下午的时候调整好了 Akismet....  是一个防Spam的插件...

情形比最初的几天已经好很多，我每天都可以零星的收到两三份留言。。中间夹杂着几份

Spam，之前一直都是手工屏蔽，不管我预感到“暴风雨”的来临。。。所以我先设置好了这份。。。感到安全许多。。

流量方面。。。我收到了三个东欧（中欧）的流量（奥地利，波兰和德国×1） 以及部分新的北美流量。。。我估计是 Google Code Jam 那篇的缘故。。。。（貌似我的博客在搜索引擎里的排名还很容易靠前的。。。）

额。。再次之前，我从来没有想到网上 Spam机器人 会这么多。。。。而且大部分都是英文的。。（俄文的）。。。令我感到很恐慌。。。 Akismet 是 Wordpress 的两个默认插件之一。。我希望她可以做到足够强大～～

嗯，也就是说我给自己昨天放了一天假。。一道题目没写。。。剩下的大部分时间也在做一些细微的改动。。。

我禁用了 JR Favicon 插件。。 这是一个设置网页 Logo 的工具，（嗯，准确的说应该不算是Logo 是网页旁边的32*32的图标 *.ico 文件。。）虽然我自己的 Logo 还没有绘好，不过我拿了那个前几天帮 Robin 的网站做的 png 做了一下实验。。。具体的效果则令我非常不满意。。。

除了浏览器兼容问题（只能在火狐下显示），另一个问题是。。只会在首页显示，（比如，当具体浏览一篇文章的时候，这个插件居然一点效果都没有。。）（虽然我还没有找到更合适的替代品。。不过考虑到反正具体的Logo也不用那么着急，所以再次之前反正也派不上用场于是考虑先禁用掉它也没有关系。）
