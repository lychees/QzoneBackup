—————— 最后这个题就变成了一个数据结构题。。。

给定 [0, n) 个格子。。。每个格子初始时 0.。。。要求支持

：向某个格子中插入一个数。。
：删除某个格子中的一个数。
：询问 [l, r] 区间的格子中第 kth 大的数是多少。。。

—————— 其中每个格子对应 SAM 中的一个状态。。。能插入的数对应这个状态中所有字符串的长度。。
注意有一个 trick 。。。“该结点里有可能有长度小于当前串长度的值” 。。。
。。由于这个范围可能是 O(n) 的。。例如。。。bbbbbbbbbbba 。。。。
因此还需要支持
：询问某个格子中，< x 的数有多少个。


主要还是区间 kth 值。。。

1. 线段树（树状数组）套平衡树
。。线段树分割区间。。。平衡树记录 值 。。。
。。每次二分答案。。。每次在平衡树或树状数组中找到对应区间。。。求这些平衡树的 rank 和。。。
。。。复杂度是 O(log^3n) 的。。。


2. 线段树（树状数组）套平衡树
。。线段树分割值域。。。平衡树记录拥有这个 值域的位置。。。
。。。然后直接 select 第 kth 大。。。 。。。
。。。复杂度 O(log^2n)


3. 线段树（树状数组）套主席树
和方法 2 一样。。
。。。复杂度 O(log^2n)

#define pp par[uu]

RST(trans[tot]);
return tot++;

CPY(trans[tot], trans[u]); par[tot] = par[u];
return tot++;

inline int h(int u){
return len[u] - len[p];

int Ext(int c){
int u = tail, uu = new_node(); len[uu] = len[u] + 1;
while (u && !v) v = uu, u = p;
if (!u && !v) v = uu, pp = 0;
else{
if (len[v] == len[u] + 1) pp = v;
else{

while (u && v == _v) v = vv, u = p;
if (!u && v == _v) v = vv;
}
}
return tail = uu;
}
