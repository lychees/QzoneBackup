1000：
给定 n 个关卡，每次闯关，你有 p1 的概率得到一颗星，p2 的概率得到两颗星。（1-p1-p2) 的概率没有得到星。
。。（重复打同一个关卡取最高。。）。。问要求所有关卡都拿到一颗星，且其中 m 个关卡拿到两颗星最少期望的游戏次数。

Solution：

。。首先。每个关卡都得拿到一颗星。。先累加这个部分。。。
DB res = 0; ECH(it, I) res += it->e1();

。。这个过程结束后，如果得到星个数不够。。我一定是从。。没有获得两颗星的关卡中。。按照 p2 从大到小的顺序去拿第二颗星。。
。。因此首先要对关卡按照 p2 排序。。

。。。问题是我们在第一步中不能知道具体哪些关卡是获得两颗星的哪些关卡没有。。
。。所以要进行接下来的一步需要先 DP。。这是我没有考虑清楚的地方。。。显然我膜拜了 CLJ 的代码。。。。


——————————

dp[i][j]：表示在 (i.. n) 这些关卡中，有 j 个关卡取到了两颗星。
（注意上面是开区间，i 关卡获得两颗星与否在 dp[i] 中是不考虑的。因为取 i 关卡的两颗星必须要求之前这里只得到一颗星。）

这个 DP 很简单。。

dp[n-1][0]=1; DWN(i, n, 1) REP(j, n-i){
DB u = dp[i][j];
dp[i-1][j] += u*I[i].q();
dp[i-1][j+1] += u*I[i].p();
}

这里的 q() 表示通关的时候只取到一颗星。p() 通关的时候取到了两颗星。。。
这玩意似乎是非常显然的。。。但是这是另一处我当时一直纠结的地方= =。。。。

再对 dp[i]。。 求前缀和 现在状态就是有小于等于 j 个关卡取到两颗星了。。
现在直接扫前 m 个关卡就行了。（后面的关卡肯定不会再用到。。）

const int N = 2009;
DB dp[N][N];
struct rec{
int p1,p2;
rec(int p1,int p2):p1(p1),p2(p2){}
bool operator <(const rec& rhs)const{return p2<rhs.p2;}
bool operator >(const rec& rhs)const{return p2>rhs.p2;}
DB e1(){return 1000./(p1+p2);}
DB e2(){return 1000./p2;}
DB p(){return (DB)p2/(p1+p2);}
DB q(){return 1-p();}
};

class DoraemonPuzzleGame {
public:
double solve(vector <int> X, vector <int> Y, int m){

vector<rec> I; int n = X.size(); REP(i, n) I.PB(rec(X[i],Y[i])); sort(ALL(I), greater<rec>());

DB res = 0; ECH(it, I) res += it->e1();

memset(dp,0,sizeof(dp));

dp[n-1][0]=1; DWN(i, n, 1) REP(j, n-i){
DB u = dp[i][j];
dp[i-1][j] += u*I[i].q();
dp[i-1][j+1] += u*I[i].p();
}

REP(i, n-1) REP_1(j, n-i-1) dp[i][j] += dp[i][j-1];

// 出现 <= m-i-1 个
m -= n; REP(i, m) res += I[i].q()*dp[i][m-i-1]*I[i].e2();
return res;
}
};
