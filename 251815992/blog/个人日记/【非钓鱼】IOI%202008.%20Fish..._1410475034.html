先整理题目。。。我们只关心某个组合能不能取到。。而不关心具体吃的过程。
那么我们只要考虑这个组合中最大的鱼即可。。。 判断某种组合存不存在是非常容易的。。

4

1 2

2 2

7

合法的组合是：1,2,11,22,12,21,112,221（。。当然。。上面 12 和 21 是重复的。。

我们发现想要避免重复计数非常棘手。。。

// 1,11

解决重复计数的一种方法是。。。。对每一种方案与其中的一个元素建立映射。。在只有枚举到那个元素的时候才会枚举到。。
。。（所谓【提纲挈领】”。。

在这个问题中。。。我们就是对所有种类的鱼。。按照 longest 排序。。。

A B
a b

然后每类只枚举比他小的。。。
这样的话对于 Input 1 。。我们会漏掉。。。A a b 这组。。。
这是因为 B 无法吃掉 A。。。
这种情况只会存在于。。A a （即最大的 A 把能吃的 a 全部都吃掉的情况。。否则的话。。后续的 B 因为比 A 大。也是能吃掉的。。）
但这种情况。。在样例 2 中却不会发生。。
这又是因为 B 已经可以吃掉两个 a 了。。。

所以我们要做的是。。。二分查找第一个可以吃掉 A 吃不掉的 a 的 B。。。。

VVI a; VI A; int cmp(const VI a, const VI b){
return a.back() < b.back();
}

int n, k;

int main(){
#ifndef ONLINE_JUDGE
freopen("in.txt", "r", stdin);
//freopen("out.txt", "w", stdout);
#endif

RD(n, k, MOD); a.resize(k), A.resize(k); DO(n){
int len; RD(len); a[RD()-1].PB(len);
}

REP(i, k) SRT(a[i]); SRT(a, cmp); REP(i, k) A[i] = a[i].back();

Int z = 0; REP(i, k){
Int t = 1; REP(j, i) t *= UBD(a[j], A[i]/2) + 1;
int aa = UBD(a[i], A[i]/2); z += t*aa;
int up = LBD(A, a[i][aa]*2);
FOR(j, i+1, up) t *= UBD(a[j], A[i]/2) + 1;
z += t;
}

cout << z << endl;
}
