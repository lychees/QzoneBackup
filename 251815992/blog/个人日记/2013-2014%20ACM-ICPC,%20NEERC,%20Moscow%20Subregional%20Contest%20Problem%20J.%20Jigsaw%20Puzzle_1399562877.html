问题描述：
。给定 n*m 的棋盘。（max(n, m) <= 6)。。。。
问有多少种对棋盘的裁剪方案。。。使得存在完美多米诺覆盖。。

分析：

分析方法类似 SRM 619 的 1000。。。我们如法炮制。。


首先弱化。。
。。。考虑给定一个棋盘，问是否存在完美多米诺覆盖。。。除了二分图匹配之外。。当然也可以状态压缩 DP。。。
考虑逐层 DP。。。s1 记录上一层是否有东西凸出来，s2 记录当前这一层的裁剪状态。。
两个如果不冲突，并后取反，可以得到本层的 mask。。。（需要覆盖）。。。一共是 2^6。。。

。。。回到原问题。。。
。。。初看我们状态一共是 2^ (2^6) 种。。。难点同样是如何化简。。。


观察。。。




。。。。我们发现我们其实只需要计数。。这两种情况并不需要加以区分。。只保留兼容性更大的后一种就可以了。。。
。。也就是那 2^6 状态里。。我们把有相邻 1 的。。全部删去。。这样就只有 Fibnacci 种了。。。

。。。。似乎也有其他的化简方法。（例如利用对称性？）。但是我们只需要用这一条。。（当然 + 各种 map 保存状态，保存转移。。是必不可少的。。。）。。就已经可以通过了。。。

const int M = 6;

map<int, Int> dp[2]; VI adj[1<<M];
int p, q, n, m;

void dfs(int i, int s, VI &adj){
if (i == M) adj.PB(s);
else{
if (i-1>=0 && _1(s, i) && _1(s, i-1)){
dfs(i+1, s-_1(i)-_1(i-1), adj);
if (i+1<m && _1(s, i+1)) dfs(i+1, s-_1(i)-_1(i+1), adj);
}
else{
dfs(i+1, s, adj);
}
}
}

map<VI, int> H; VVI L; int h(VI &s){
if (!CTN(H, s)) H[s] = SZ(L), L.PB(s);
return H[s];
}
