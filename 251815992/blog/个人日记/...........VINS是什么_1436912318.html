利用一个相机和一个IMU整合的系统确定系统在空间中的位置、姿 态和速度。实现在无GPS的环境中定位。
令人惊奇的是，这类算法看似要用到复杂的图像处理和机器视觉等知识，但是其实图像和机器视觉算法在整个系统中并没有非常显著的地位。视觉和惯性导航的融合主要还是强调filtering的robustness。According to Shaojie，就算图像之间的特征匹配效果非常差，足够稳健的系统还是能够给出不错的位置估计。

目前比较前沿的工作：

https://fling.seas.upenn.edu/~shaojie/wiki/index.php?n=Main.HomePage
https://www.youtube.com/watch?v=jIMSkgliQrk

这个领域的研究人员很少，做得优秀的自然更少。

学习方法：​

理解Kalman Filter是关键

Probabilistic Robotics和Linear Estimation肯定是必读的。
不过都不用一下读太多。

LE读以下章节:
2.1 2.2 2.3
3.1 3.2 3.3 3.4
4.1 4.2 4.3
9.1 9.2
PR只需要先读前三章，特别要仔细理解Extended Kalman Filter


接着学习IMU姿态解算的算法，先自行google/百度加速度计原理陀螺原理磁感计原理，MEMS元器件特性等知识。

接着自己购买IMU开发板（淘宝上一搜一大把），我推荐 http://chiplab7.taobao.com/ 的各类产品。附带有完善的教学资料，以及一些技术算法的介绍。把这些教学资料都弄明白基本就懂了IMU和四轴飞行器了。

接着购买摄像头，通过opencv熟悉常用的图像处理算法和机器视觉算法。

接着学习ROS，利用ROS的framework实现IMU数据和图像数据的读取甚至同步。

接着制作一个开发平台，便携、稳定、具有一个相机和一个IMU，还有足够的计算量。

然后要精读以下几篇文章：
1. Indirect Kalman Filter for 3D Attitude Estimation
http://www-users.cs.umn.edu/~trawny/Publications/Quaternions_3D.pdf


2. Stochastic cloning: a generalized framework for processing relative state measurements
或者
SC-KF Mobile Robot Localization: A Stochastic Cloning Kalman Filter for Processing Relative-State Measurements

3. A Multi-State Constraint Kalman Filter for Vision-aided Inertial Navigation

4. Shaojie Shen Phd Thesis第6章

读的过程中纸笔和MATLAB不能离手……
