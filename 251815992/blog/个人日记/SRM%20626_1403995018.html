。给定一个长 sideA 宽 sideB 的矩阵。。
从左下角发射一枚光线，要求恰好在墙壁反射 n 次后落入右上角，且中途不能经过 corner。。。


Analysis:

首先 n 必须是偶数。。
设在横轴与纵轴分别反弹 a, b 次
那么平铺后的长宽分别为 (a+1)sideA, (b+1)sideB。。。
a+b == n，gcd(a+1, b+1) = 1
——————


然后显然是一个莫比乌斯反演。。
。。。

s2(n) = 1^2 + 2^2 + \ldots + n^2 = n*(n+1)*(2*n+1)/6 ...
g(d) = [d | gcd(i, n-i)] * (i^2)  =  sqr(d) * s2(n/d);
f(d) = [d = gcd(i, n-i)] * (i^2)
...
