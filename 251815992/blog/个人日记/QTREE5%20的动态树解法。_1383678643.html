这个题的动态树解法开始并不直观。。。先回归 qzc 原论文。。。






。。一图流。。。。还记得这是什么吗？。。
。（把轻重边树链剖分开成链分治。通过维护前/后缀和来得到过这条链的某个区间的最近白色点对。。）
。。。树链剖分里我们是通过线段树 （ 区间合并） 来维护树链 。 。通过堆来维护叶子节点的。。。。

。。而在动态树里。。我们是通过 splay 来维护树链。。。每个节点再外挂一个堆。。。
。。来维护根节点从所有虚孩子那里得到的信息的。。。。
。（现在我们知道了。。这种需要支持删改的堆可以偷懒用 multiset 。。ww。。）



。。。。。。没错。。其实就是我们的三叉子树动态树。。。（参见 QTEE7 的一种解法。。。
http://user.qzone.qq.com/251815992/blog/1382108682


改两个函数就行了 ww。。。（Dalord 再也不用担心我调不出轻重边树链剖分了ww）
inline void upd(){
sz = l->sz + 1 + r->sz; int m0 = min(w0, *s.begin()+1);
w1[0] = min(l->w1[0], l->sz+min(m0, r->w1[0]+1));
w1[1] = min(r->w1[1], r->sz+min(m0, l->w1[1]+1));
}
inline node* acs(){
node *x = this, *y = NIL; do{
x->splay();
if (y != NIL) x->s.erase(x->s.find(y->w1[0]));
if (rx != NIL) x->s.insert(rx->w1[0]);
rx = y, x->upd();
y = x, x = px;
} while (x != NIL);
return splay();
}

注意 erase 的时候要用 find 。。。（erase val  居然是把值等于 val 的全部删掉。
