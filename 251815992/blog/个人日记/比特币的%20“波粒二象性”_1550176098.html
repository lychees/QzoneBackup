# 比特币的 “波粒二象性”

比特币是一束光。

光具有波粒二象性，而比特币具有货币股份二重性（Currency-Share Duality）。

书上说，[货币天然是黄金]( https://www.azquotes.com/quote/709805 )。但今天，被比特币启蒙过我们应该已经认识到，货币天然是信任。CSW 和 Roger 早年在 BCH 还没有分家的时候，逢场必说的一个观点，便是强调 Bitcoin 的货币属性。Bitcoin 应该一种被日常使用的 Cash，而鼓励人们， [Use it rather then HODL]( https://youtu.be/95ZhGrywBuo?t=45 )，只有被使用的 Cryptocurrency 才有价值。

今天且不说 [Bitcoin 的 Blocksize 实际已经远大于 BCH 和 BSV]( https://twitter.com/kerooke/status/1096105643209969666 )，就作为货币的层面而言，前者也因为 [Lightning Network 的日趋成熟]( https://mp.weixin.qq.com/s?__biz=MzUxMTcwNDQzMA==&mid=2247490149&idx=1&sn=786fe4941e6d32678155b3c5787b3fe8&chksm=f96ef664ce197f7221aec8c7d31df44561eef364caa35e026d9f0568a645775d29b2e0246725&mpshare=1&scene=22&srcid=0213ltYznIzqE6Dc7dGJTUVc#rd )，而变得更像是一种货币。比特币具有其货币属性的一面，但一味的强调其 “货币” 属性，令 Roger 等人忽略了其另外一面，比特币作为 “股份” 的一面。

作为股份的 BTC，希望让人囤积。
作为货币的 BTC，希望让人使用。

作为货币来说并不适合价格不断上涨，而是温和通胀。
但作为股份，不断通胀没有人会愿意持有，股份需要的是人们跟随项目持有并增值。
而比特币的增值来源的核心因素之一又恰好是有人不断的去使用她，也就是其货币性。
人们持有比特币，并期待着有一天其能够成为 “世界货币” 或者 “The Currency of the Internet”。

这种看似矛盾的特性在比特币中是相融，共存的。

当然，不仅比特币，所有的 Token 都或多或少的具有这种 Duality。
一个更容易理解的例子是 [BNB]()，作为币安的平台币，BNB 代表这币安的现在与未来（Binance Dex and Binance Chain），同时，[币安一直在努力让更多的渠道和平台直接支持 BNB 支付]( https://twitter.com/cz_binance/status/1094836657159692288 )。根据福布斯报道，[一些 “新创企业” 也已经开始使用 Token 建立其新型的组织形态，零雇员公司或者去中心化自治组织]( https://www.forbes.com/sites/geraldfenech/2019/01/25/interview-how-this-startup-is-building-zero-employee-organizations-using-blockchain-technology/?from=timeline&isappinstalled=0#288a295e74af )。

无法理解比特币就无法理解 Token，自然也设计不出相容的 Token Economy。Token 将会重塑未来的经济形态和组织形式，not through something like STO，but will be something beyond the stocks.

“Let there be light!”
