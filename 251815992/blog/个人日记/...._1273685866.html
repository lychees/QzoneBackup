先介绍一下我手边的几个工具。。。

互联网上有很多有用的站点，也有很多只是引用人家站点的没有用的站点....

但是不管怎么说所有构成这些站点的部分都会是一些标签结构的HTML文本文件（ 有迹象表明，HTML5时代正在悄悄像我们靠近 。）（编辑查看网页源代码就可以得到它们）

不同的浏览器 ( Broswer ) 对这些文本文件进行解释，解释成我们现在看到的画面，所以可能会出现这样的情况也不是不可能.... 例如两个不同的浏览器所翻译成的页面不一样。。。

但是我估计浏览到这篇的同学用的会是 IE 浏览器，嗯至少之前我用的腾讯TT的浏览器，因为单击QQ空间会默认用TT浏览器，或者也有可能是 安全卫士的安全浏览器 ，但是用这些浏览器的时候并不会发生看到页面不一样的情况，因为这些浏览器用的都是 IE 的内核。

一些网站对非 IE 浏览器的支持并不是很好，例如QQ空间，如果我用火狐打开它的话会无法播放背景音乐。 但是也存在一些网站对 IE 的支持非常糟糕。。例如UVA。。。
