A. 拓扑排序

B:  DP / 容斥
好题。

暴力冗斥 O(n4)

REP_1(i, N){
REP_1(j, N){
A[i][j] = pow(2, i*j) - 1;
REP_1(ii, i) REP_1(jj, j) if (ii != i || jj != j){
A[i][j] -= A[ii][jj] * C[i][ii] * C[j][jj];
}
}
}

官方题解是 O(n3) 的。。
dp题，我们一行一行的考虑。dp[i][j],表示前i行，都满足了每一行至少有一个宝石的条件，而只有j列满足了有宝石的条件的情况有多少种。枚举第i+1行放的宝石数k，这k个当中有t个是放在没有宝石的列上的，那么我们可以得到转移方程： dp[i+1][j+t]+=dp[i][j]*c[m-j][t]*c[j][k-t]
