题意：数轴上分布着 n 个村庄，第 i 个村庄每个时刻因疫情将会死亡 a[i] 个人。
你从村庄 1 开始，向村庄 n 行动，每个时刻你可以移动一格或治疗当前的村庄。
被治疗的村庄以后不会再有人死亡。

如果第一次经过时跳过了某个村庄，那么下次往回走时，
必须经过这个村庄并且治疗后才能继续往前走。

问最优决策下，所有村庄最少会死亡的人数。

倒着 dp。
f_{i, j}：表示从 i 开始，且会在到 j 的时候返回 i 的最小收益。

设 s_i = \sigma j >= i a_j
转移：
f[i][j] = f[i+1][j] + min(s_{i+1} + 3s_{j+1} + 3(j-i)a_i, 2s{i+1}+2{j+1})
// 回来的时刻再治疗，立即治疗。
以及 f[i][i] = min{f[i+1][j]}  + 2s_{i+1}
