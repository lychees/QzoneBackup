今天看到一条微博啊，说公司 CTO 对公司制度不满啊，失联之后敌视员工，老板亲自跑来上海，还啥都没解决啊。国内开发者怎么会有这种傻逼败坏风气啊。

非常不巧的就是，我就是那个傻逼啊。虽然说我也已经过了被骂之后就得骂回去的年纪，不过这事吧，本来就觉得发生地太过恶心，没必要全摊开说以免恶心别人，特别是当事人年纪还特别小，说别人不好显得自己气量很小，结果竟然先被挂了。不如我来说说这事有多扯淡。

首先这公司啊，开了一年多，融了一轮 pre - 天使，干了一年多，啥产品都没做出来。如果不是这人有个影响力不错校友还有个做人还行的投资人给他说说好话，我也不至于说推掉了几笔价值不菲的大单子，砍了自己一半以上的工资，不谈好股权细节，就过来帮忙接管这个团队。要说自己要点什么，我见过的钱比这家公司下一轮的融资还多，不如说是希望能做出点什么，不要把好好一件事给做坏了。

我在公司的第一天就发现公司连自己做什么产品的思路都非常混乱，聊到凌晨总算把产品路线全搞定了，最终决定重构整个代码。在此之前，每个开发者都能自己 merge，没有工作流程，没有验收流程，以至于连做成什么样都没有一个人知道。开始重构的一个星期，git flow 也正常了，集成持续也有了，连 code review 都有了。以至于原来一直落后的技术都赶到产品和设计之前去了。

当然为了达成这样的目的，我也是做了很多限制。CEO 不要再插手技术，不能打断别人工作，不能干扰别人私人生活时间。这件事情再三确认和保证，结果工作第三周就搞出大新闻：自己上去部署系统，把 nginx 给部署挂掉。一天来来回回找我要这个要那个，把我一天工作打断了四五次，最后我实在忍不住，我当天工作就先暂停了，打算第二天再说。结果这个夺命连环 call 一直打到大半夜，并第二天一早说再不来就把我换了。那我说你换人吧，就把公司项目权限移交了，然后自己的账号删了。

这事做得有点太狠当然是我气得不行。后来公司的前端开发、客户端开发一个个找我聊，说先缓两天回去劝劝老板。结果老板诚心诚意接受道歉，一个开发说如果可以明天和我一起聊一下见一面，本来这事也算好说话。谁知道这个道歉电话，竟然又是凌晨 2 点用上海公平路北外滩桔子水晶酒店的电话打过来的，我当场脾气就炸了。如果说这不是虚心接受屡教不改的话，这种人就只能叫傻逼了。

既然说要开公司，作为一家公司，企业文化可以说是随性的，老板也可以说是想怎么玩怎么玩的，但是说对于工作上的事，连自己在做什么都无法认清，那必然是无法成事的。不过反过来说，这家公司的老板也是太年轻，甚至注册公司的时候都没有满 18 岁，要让父母当公司的法人。但是，你的合伙人你的员工不是你的父母，没有理由会像父母一样迁就你。投资人之所以会迁就你是因为如果再不拉你一把，他们的钱就会打水漂。而公司一年做不成什么产品，技术负责人走了一个现在又走一个，招来的员工，来一个走一个的，还要把遇到的问题归结到大环境上，说开发者风气浮躁，倒不如看看是不是自己浮躁比较好。

有时候觉得，每个人虽然不一定要上大学，但每个人都要受教育。每个人在说什么之前最好知道世界是什么。否则也不至于强行要说自己是 dev 出身，在服务器上折腾一年还能把 nginx 搞挂掉；把服务器所有依赖都停留在有安全漏洞的版本上；公司代码文件传输不上 https；连自己房间路由器部署个自动翻墙都弄得路由器死机；还没事对着公司技术指手画脚。除了 naïve 之外，我也不知道说些什么好。

我只能说，投资人现在这有钱瞎投项目啊，毁了的不止是自己的钱，还毁了一个高中还没毕业的学生的认知，让一个人盲目，最终毁了这个人的一生，这才是这个环境的问题。
