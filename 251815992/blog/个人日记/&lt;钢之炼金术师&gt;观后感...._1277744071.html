... 应该还有一集就会完结了, 傍晚时候看完了No.63...

(其实62也是在去安庆的那几天里看完的, 我还可以记得当时的感想... 觉得是比电磁炮差一些吧, 恩~~也不能这么说...)

结局来的太突然了，( 对我而言...)

好象忽然之间 那个本远远凌驾于炼金术师上的"真理"就消失了~

( 虽然设定想一想也是蛮"不科学"的东西, 不过我很喜欢那种不管自己所面对的对手是多么强大 ( 恢复弟弟的身体)

都在之中不断从中寻找"线索"的钢之炼金术师~ 另外我期待作者最后会在什么地方收笔...

... 七个人造人全死 .. "父亲"大人也挂了... 弟弟的身体被换了回来 ( 没有用贤者之石...)/....

...

给我留下印象最深的是大佐烧死拉特斯时候的剧情.... ( ... 居然..第一个死掉的人造人会是她....)

( 还有第一集给人留下的印象也很深... 用这种方法交代后文所有用到的世界观虽然有点直接呀不过很容易令人接受...)

还有第一次斯卡和爱德华打架的剧情...( 特别是一掌轻轻一推把机械臂打碎掉的画面......)

... 到是之后斯卡的几次战斗却没有第一次那么深了....

但是这些都不是最好的地方....

我最喜欢的地方是在中间的十几集里...

如果人物里只有艾尔利克兄弟一条线的话... 就不会是这样了...

正是因为在这个世界观存在这好几支势力....

艾尔利克兄弟...大佐..红莲..大总统..马鲁克医生...人造人....斯卡.!.....姚....张梅.................

而人物之间的关系网很复杂但是性格却都很清楚....

(像是古利得....哦...人物的名字有的时候直接用性格里可能的英文名转写了..)

到后来他们之间不同的几次交手... 这段人物间互相分开写的部分是我记忆里最深的..

下面写"败笔"... (额, 只是我认为写的不好的地方... 似乎只要不合我的心意就会被我认为是写的不好了呀...)

` 金伯利 ...

首先.. 这个人物在第一集里就有出来...中间也一直做伏笔...但是到好后来才放出来...

我觉得他同Scar的那次交手 (被远距离攻击命中了.....) 和死掉的那一场 ( 被狮子咬断喉咙给秒杀了...)

的都很出人意料!.... 但是我觉得最可惜的是最后死掉以后又出来干扰Pride的那一场....

.... 我个人是灰常讨厌那一段剧情的...

特别是那句... "你根本不了解  爱的华`艾尔利克...."

....这句话太假了....而且他之前同爱德华认识的剧情也好少啊....( 只有北方的那几次会面而已.... 战斗也只有一次...)

`Scar...

我觉得Scar和Greed是写的最好的两个人物...

但是Scar在最后几集里什么剧情都没有...而且他同大总统的那场战斗太血腥了....

`.阿尔逢斯

....在第一集的时候甚至觉得这个人物有点似乎是为了写他哥哥而硬加的...

不过看到后面我也想过很多。.. 包括他最后为了击败人造人而献出自己的灵魂换回哥哥的胳膊的剧情都很感人...

但是最后他哥哥把他救回来的剧情太假了。..( ...居然献出的是自己的真理之门...并且..他还说..恭喜你..答对了..- -)

(反正是有一点"失望:"呢也许是我想的太多了...总觉得这里应该不只是这样... ...可是要怎么写呢? 要怎么写才能令我."满意"呢....要怎么写呢?...... )
