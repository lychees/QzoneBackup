首先写出状态转移方程。。


checkMin(dp[u], dp[pp]+ss[u]+vv[u]*(dd[u]-dd[pp]));  ... 其中 pp 是 u 的某个祖先。。

。。分离常量。。
...dp[u] = min{dp[pp] - dd[pp]*vv[u]} + ( ss[u] + vv[u]*dd[u])


考虑如果 p1 决策比 p2 优。。。整理可得。。。

vv[u]  < dp[p2] - dp[p1]
---------------
dd[p2] - dd[p1]

我们维护这个凸壳。。
由于 边长都是正值。。。 dd[pp] 单调递增。。所以属于第二类斜率 DP 问题。。可以直接用    栈   维护凸壳。。。
。。。



剩下的问题就是。。。退回父亲节点的时候如何简单维护的实现上的问题了。。。。。




类似十字链表中的做法。。。
考虑我们当前栈中的元素是 。1,2,3,4,5,6,7 。。。。
。。。插入的新元素在 3 位置。。那么 4,5,6,7 本应该是被删除的。。。

但是为了可以复原。。我们并不执行删除。。。而是在将新元素插在。。一个新的位置。。并和 3 位置交换。。。。。
。。记录此时栈实际的深度。。。。

。。返回的时候。。交换回来即可。。。

int stk[N]; DB slp[N]; // 左边表示 栈 中结点的编号。。右边表示斜率。。。

#define k(x) ((DB)(dp[stk[x]]-dp[u])/(dd[stk[x]]-dd[u]))

LL dp[N]; int stk[N]; DB slp[N];

void dfs(int u = 1, int p = 0, int rr = 0, int tt = 0){

int ll = 0; while(ll < rr){
int mm = ll + rr >> 1;
if (k(mm) < slp[mm]) rr = mm;
else ll = mm+1;
}

// 单峰 。。

stk[tt] = u, slp[tt] = ll ? k(ll-1) : 0;
swap(stk[ll], stk[tt]), swap(slp[ll], slp[tt]); ++ll; ++tt;

REP_G(i, u) if (v != p){
dd[v] = dd[u]+w; int pp = stk[lower_bound(slp, slp+ll, DB(vv[v]))-slp-1];   // 二分。。
dp[v] = dp[pp]+ss[v]+vv[v]*(dd[v]-dd[pp]);
dfs(v, u, ll, tt);
}

--tt; --ll; swap(stk[ll], stk[tt]), swap(slp[ll], slp[tt]);
}
