考虑用 manacher 处理出了每个位置的回文半径 r[i]。。
那么对于每组询问 (l, r)，只需要统计..
\sigma_{i=l}^r min(r[i], i-l+1, r-i+1) 即可。

（注意 manacher 中我们加了额外的字符。。要把询问的区间映射过去。。）
（最后算答案时再去掉多出来部分即可。。）

Rush{
int l, r; RD(l, r); l = l * 2; r = r * 2; --a; ++b;
LL z = 0;
for (int i=1;i<n;++i){
z += max(0, min(::r[i], i-l+1, r-i+1));
}
z -= (r-l)/2+1;
z /= 2;
}

注意到可以分类讨论去掉 min 里面的一个条件。。
再离线用两个 树状数组维护即可。。
