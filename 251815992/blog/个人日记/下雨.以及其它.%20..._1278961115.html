我输入法坏了... 现在勉强能用智能ABC....( 好吧, 我的机子是出了一点小故障... 大概明天要修机子了....)

于是输入法坏了让我都不是太想写东西 ( 借口..绝对借口....)

....

所以我非常抱歉 旅行的记录文 放不出来了.... 只好 找几个有图片的帖子了 ...

.... (再写这篇的时候又来了 一个会动的 ... PS: 真没技术含量.... )

....

1.

于是那天那个 "绿皮拖拉机" 下到沼泽里引来全镇的人围观是最后的高潮了... 但是我对这种烧钱啊的活动是多么讨厌啊..

其实在之前安排旅馆去的那一次... 后来在图书馆里的那几天.... 我已经想 写一篇表达我的立场 了...  那篇其实措辞很伤人... 于是为了不让我爸看懂.. 特地写成了英文.... 并且我之前的那位 Wilbert 同学还答应帮我订正语法和拼写错误... ( .... 确实有几个很白痴的拼写错误...)

但是... 最后的情况是比较惨的... 写到后面的逐渐我失去了控制... .. 而且居然最后的篇幅也逐渐变的很长起来...

然后看的她很不爽..一皱眉干脆隐藏起来了...

2.

哎... 然后还是以前的文字比较耐读些呀....

发现最近出现这种情况还不止有一次啊...  算了反正都过了那么多天了...把 这一篇 也解除隐藏好了....

反正是写了就忘...

3.

之前的那篇 游记  最后没有总结掉... 如果是完整写完的话最后我是要写的是那几天里我所有的感想...

包括值得感动之事... 很无法接受之事... 以及为什么最后我不高兴...

... ( 其实外出是非常疲惫之事... 果真是只能同自己的说的来挚友一起才好....)

还有另外一点则是... 我没有相机的控制权....  而我现在的手机又不能拍照。..

但是这一次是不一样的... 总之虽然我还是必须同他一车... 但是起码我可以有相机的控制权...

但是我们不住一块.... 我却不能把她带回来发图...... 这算什么说法啊??.........

4.

前几天安徽虽然是阴雨连绵的天气。.. 导致 最后所说的篝火晚会之事都没有办成 ...  但是我是很喜欢那样的天气的..

... 路过花亭湖的时候... 经过了一段山路... ( ..我拍了N张非常好看的图.... 因此我认为他们之前拍的之所以差的原因是因为画面没有层次感....) ...

那种红色的一片一片.. 并不是另一种林木..  而是松线虫病.....

5.

在山里看到两只棕色的小野羊.... 好想抱回家....

... 但是旅途中的大半行程中间是不能停的.... 所以这种感觉有点想有些地方虽然经过很多次... 但如果不能步行下去探索的话...还会感觉很陌生.... (有点新快三里面第三关的感觉- ....#>#>#>#>. 我每次都是一开始就接电台的... 你在说什么? 听不懂就算了吧...)

6.

遭遇了一场车祸... 差点挂了........ ( ... 记述文好难写.)
