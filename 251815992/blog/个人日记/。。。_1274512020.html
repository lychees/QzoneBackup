我在 阅读这篇 的时候知道这个。。。 The GNU Multiple Precision Arithmetic Library 。。。

。。。GNU。。多倍精度运算库。。。

。。。

先简单介绍这个组织和这个开源库。。简单的说他们是要实现在任何情况下都可以做到无限精度的各种大数据的运算。。。同以往的 Project 不同。。这个项目最看重的是类库的运行时效率。。（甚至不惜调用各种汇编。。）

当然本篇并不旨在介绍GNU。。。而是在于介绍GMP。。。

即便不是用GMP。。去研究它的内在实现（What happend behind the scene）也可以从之中学习到非常多有意义的知识。。

从这个项目不仅可以看到诸多灰常实际的内嵌汇编的应用。。还可以收集许多具有应用价值的数学知识～例如 Karatsuba multiplication ， Binary GCD 以及比上一篇所描述的更具有破坏力的平方根算法。。。（ 甚至已经有一份 Nth Root Algorithm ...）
