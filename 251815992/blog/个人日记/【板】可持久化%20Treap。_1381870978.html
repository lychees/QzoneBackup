（合艾赛区是哪个赛区？）

split(x, p, a, b)  （将 x 为根的子树，分裂成 a（前 p 个元素）、b （剩余元素） 两个子树。。

如果 p <= sz[lx] 。。说明 a 完全包含在 lx 中。。递归到 split(lx, p, a, b)。。 b 子树由剩余部分组成。
反之。。。。。。。说明 b 完全包含在 rx 中。。递归到 split(rx, p-sz[lx]-1, a, b)。。a 子树由剩余部分组成。

void split(int x, int p, int &a, int &b){
if(!p) a = 0, b = x; else if(sz[x] == p) a = x, b = 0;
else{
x = new_node(x);
if(p <= sz[lx]) split(lx, p, a, b), lx = b, b = x;
else split(rx, p-sz[lx]-1, a, b), rx = a, a = x;
update(x);
}
}
