Brief description:

给一个不自交的简单多边形，上面有一些洞，问最多能装多少水。连通块的水面不能高于任何洞。

solution:
http://hi.baidu.com/lccycc_acm/item/7a82c0e1ff26a8f22a09a4af ;

“这确实是看了就不想写的题目，但是我想出了一个优雅的解法。先将多边形按 y 坐标离散切了，于是切出来的全部是梯形。然后从低到高通过并查集来维护连通性，同时维护这一块连通块内部有没有洞。每次处理一层，然后看每个新加入的梯形，如果所在连通块有洞就不累加面积。这种方法代码很好组织。注意“喷泉”的情况。 ”

好组织个毛线啊。。。

直接在扫描线的同时切梯形。。。我们用一个 map<Po, VI> H 记录每个点关联哪些边。。
。。。每次处理出相邻的两条切割线所对应的点对。。。

上线：L[p]
下线：L[q]

点一定是成出现的。。（三角形作为退化的梯形处理）。。。
接下来考虑维护联通性。。。





。。。不妨我们对偶一下。。。把梯形的边看成 点。。。把连接上下边界的梯形本身看成 边。。。。。
。。于是就是从下到上维护这颗树的连通性的同时。。累加边权。。
这样可以非常自然的处理【喷泉】。。。


----UPD -------- 修正一下。。。还是得把梯形看成点。。。两个梯形上下边界有交。。看成 边。。。


然后扫描线的过程就是。。。

切梯形（切的同时处理下面的缺口），处理联通，计算面积，处理上面的缺口


void scan(){


vector<rec> L[2]; int p=0, q=1; CLR(L[p]); DB res=0;


REP(i, SZ(I)-1){

swap(p,q), cut(L[p],I[i],I[i+1]); //cout << I[i] << "---" << I[i+1] << ": " << DSU::n << endl;

ECH(hi, L[p]) ECH(lo, L[q]) if (cov(MP(hi->l.fi,hi->r.fi),MP(lo->l.se,lo->r.se))) DSU::Union(hi->id, lo->id);

ECH(it, L[p]) if (!DSU::ho[DSU::Find(it->id)]) res += it->s()*(I[i+1]-I[i]);

ECH(it, L[p]) if (ck(it->l.se,it->r.se,I[i+1])) DSU::ho[DSU::Find(it->id)]=1;

//printf("%.2f\n", res/2);

}


OT(res/2);

}
